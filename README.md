📝 Lista de Tarefas

Aplicação simples de lista de tarefas (To-Do List) desenvolvida com JavaScript, HTML e CSS, com funcionalidades completas de CRUD e persistência de dados no navegador.

🚀 Funcionalidades
✅ Adicionar novas tarefas
📝 Editar tarefas existentes
❌ Excluir tarefas
💾 Salvamento automático com LocalStorage
🎨 Interface estilizada com CSS
🧠 Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage
💡 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end, como:

Manipulação de DOM
Eventos (click, interação com usuário)
Estruturas de dados (arrays)
Persistência de dados no navegador
Organização de código
